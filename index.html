Bug Tracker

Bug Tracker is a web application designed and developed using microservices architecture. Bug Tracker is a bug and issue tracking system that helps software teams record, and track features, enhancements, and bugs in their software.

The system provides an overview of the project, authorized users, and facilitates authorization of the resources related to the project. The system provides users to create, assign, prioritize as well as track issues.

Bug Tracker is developed using technologies namely

Frontend Tools:
HTML, CSS, Bootstrap, JavaScript, React, Redux.

Backend Tools:
Java, Spring technologies (Boot, Cloud, Security, Data JPA, AMQP, Sleuth), Lombok, and more.

Other Tools:
Maven, Docker, Kubernetes, Postgres, Redis, RabbitMQ, JSON Web Token (JWT), log4j, Jib, Zipkin, Git, and more.

COVID-19 Forecaster

COVID-19 Forecaster is a forecasting system that predicts the number of deaths, confirmed cases, and recovered cases in Australia and its states. COVID-19 Forecaster uses Artificial Intelligence and data from the past to compute future predictions.

COVID-19 Forecaster is developed using technologies namely
HTML, CSS, Javascript, Python, Bootstrap, Plotly, Pandas, Numpy, Matplotlib, Tensorflow, Sklearn, and Github.

TechQuiz
