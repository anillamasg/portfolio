Bug Tracker

Bug Tracker is a web application designed and developed using microservice architecture. Bug Tracker is a bug and issue tracking system that helps software teams record, and track freatures, enhancements, and bugs in their software.

The system provides an overview of the project, authorized users, and facilitates authorization of the resources related to the project. The system provides users to create, assign, prioritize as well as track issues.

Bug Tracker is developed using technologies namedly
HTML, CSS, Javascript, Bootstrap, React, Redux, Java, Spring Technologies like Boot, Cloud, Security, Data JPA, and AMQP, PostgreSQL, Docker, Kubernetes, RabbitMQ, Redis, Maven, JWT, Jackson, and Lombok.
